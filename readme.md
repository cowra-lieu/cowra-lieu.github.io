# 我的笔记仓库

## 目录结构说明

```text
/
├── .github/                <-- 隐藏目录，用于 GitHub 配置
│   └── workflows/
│       └── deploy.yml      <-- 自动化部署脚本
├── docs/                   <-- 存放你所有的 .md 笔记文件
│   ├── index.md            <-- 网站首页（必须有）
│   ├── 编程/
│   │   └── python.md
│   └── 工作/
│       └── 汇报.md
├── mkdocs.yml              <-- MkDocs 的主配置文件
└── .gitignore              <-- (可选) 忽略一些本地临时文件
```
